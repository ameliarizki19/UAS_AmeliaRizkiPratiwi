# Campus Recruitment Prediction

Project ini bertujuan untuk memprediksi status placement mahasiswa 
menggunakan metode Logistic Regression dan Decision Tree.

## Dataset
Dataset berisi data akademik dan employability mahasiswa.

## Model yang Digunakan
- Logistic Regression (Accuracy: 84%)
- Decision Tree (Accuracy: 79%)

## Kesimpulan
Pengalaman kerja sebelum lulus dan IPK merupakan faktor utama 
yang mempengaruhi keberhasilan placement mahasiswa.
